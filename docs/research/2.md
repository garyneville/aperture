10. METAR value extraction The 4 most photographically useful fields to parse from raw METAR reports are:

    Significant Weather Observations (wxcodes): This explicitly differentiates between Fog (FG), Mist (BR), Haze (HZ), and Smoke (FU). This distinction is critical because radiation fog creates moody, diffused light, while smoke and haze consist of smaller aerosols that aggressively extinguish blue light and flatten contrast.
    Horizontal Visibility: This is a routinely reported, near-real-time proxy for local aerosol extinction, haze, and fog density that far surpasses the spatial coverage of dedicated aerosol networks.
    Temperature and Dew Point: Extracting both allows you to compute the dewpoint depression. As the temperature and dew point converge (a small spread), the boundary layer nears saturation, making it a primary predictor for radiation fog onset.
    Cloud Ceiling / Base Height: Tracking cloud bases allows you to predict if existing low stratus will "build down" to the surface to create fog. Additionally, for storm photography, low Lifting Condensation Levels (LCL < 1000m AGL) mean that storm structures like shelf clouds and wall clouds will appear much closer to the horizon and be visually spectacular.
